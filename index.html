Animation

I don't remember every animation Dr. B did, so I'll try to write about the ones that I remember.

Transition

First, I think Dr. B used .transition() to make a changes visible. With .duration() to specify the duration for the transitions. Because all the transitions of each score change occurs in about the same time, I don't think Dr. B used .delay(). To make the transitions smooth, he used .ease() and maybe with d3.easeLinear
I don't remember if there is any special animation at the beginning and end of transition.
Dr. B probably used mouseover and mouseout to show the score on hover.
